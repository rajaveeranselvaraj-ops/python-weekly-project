Titanic Dataset Analysis using NumPy and Pandas
📌 Project Overview

This project analyzes the Titanic dataset using Python, Pandas, and NumPy.
The objective is to explore passenger data and find survival patterns.

📂 Files Used

train.csv – Training dataset with passenger details

test.csv – Test dataset

gender_submission.csv – Survival predictions file

project.py or Project_2.ipynb – Python code file

🛠 Technologies Used

Python

Pandas

NumPy

⚙️ Features Implemented

Import CSV files using Pandas

Handle missing values (Age column)

Calculate survival percentage

Group survival rate by gender

Find highest and lowest fare using NumPy

▶️ How to Run

Install required libraries:

pip install pandas numpy

Run the Python file:

python project.py

OR open the Jupyter Notebook:

jupyter notebook
📊 Sample Analysis Performed

Total number of passengers

Survival percentage

Survival rate by gender

Maximum and minimum fare

🎯 Conclusion

The analysis helps understand survival patterns in the Titanic dataset using basic data analysis techniques with Pandas and NumPy.
